# Mobile UI Design Guidelines (Reference)

## Core Goal
Design a mobile-first UI that feels human, calm, and professional.
No visible AI aesthetics.

## Layout
- Single-column layout
- Centered content
- Comfortable reading width
- No side panels on mobile

## Typography
- Large, readable base text
- High line-height (1.6â€“1.8)
- Minimal font size variation
- Short paragraphs

## Buttons
- Tall, thumb-friendly buttons
- One primary action per screen
- Calm, neutral colors
- Subtle press feedback

## Chat Experience
- No chat bubbles
- Document-style text blocks
- Generous vertical spacing
- Smooth auto-scroll behavior

## Motion
- Fade-in content
- Slide-up modals
- No flashy or playful animations

## Tone & Feel
- Quiet confidence
- Human, product-led
- No futuristic or robotic styling
